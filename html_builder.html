HTML Template Builder — Drop-in Notes

Single-file, offline builder with logo upload, text navigation, rich text body, image gallery, live preview, and export to standalone HTML/JSON.

How to Use

  1. Set project basics: Enter a site title and choose a header layout.
  2. Upload a logo: PNG/SVG recommended; automatically constrained for consistent sizing.
  3. Build the menu: Add items, edit labels/links, and reorder with the up/down controls.
  4. Add content: Type in the editor or use quick-insert (H1, H2, paragraph, list, quote, rule).
  5. Insert images: Upload to the gallery, then click an image to insert it at the caret.
  6. Preview: Live preview updates as you type; no external dependencies.
  7. Save/Export: Save JSON (for later editing), copy HTML to clipboard, export standalone HTML, or print/PDF.
  8. Autosave: Uses Local Storage (browser-local only) with manual save/load/clear controls.

Implementation Notes

  • State model: Title, layout, navigation array, body HTML, and gallery (data URLs).
  • Editor: contenteditable div with simple element insertion and caret-based image placement.
  • Images: Read via FileReader as data URLs; inserted directly into the body.
  • Persistence: localStorage key for autosave; JSON import/export for portability.
  • Export: Generates a self-contained HTML file with inline CSS and your current content/layout.
  • Accessibility: Labels, ARIA roles, and keyboard-reachable controls for core operations.
